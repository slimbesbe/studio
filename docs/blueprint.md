# **App Name**: Inovexio PMP Exam Simulator

## Core Features:

- User Authentication: Secure user authentication with email/password, password reset, and 'remember me' functionality.
- Question Filtering: Access questions/exams filtered by Process Group, Knowledge Area, Exam Domains, Approach (Predictive/Agile/Hybrid), Difficulty, and Topic.
- Full Exam Simulation: Simulate complete PMP exams with customizable duration and a full question set, mimicking the real exam environment.
- Performance Dashboard: Display user's progress with overall progression bar, stats by domain/process, exam history, and a 'next best practice' recommendation engine.
- Kill Mistakes Review System: Review system using spaced repetition (1d/3d/7d) including filtering recently/repeatedly missed, by knowledge domain, with recurring errors indicator, ensuring the PMP candidate masters areas where errors are common.
- Admin Content Management: Super Admin dashboard for user, question, and exam management; including CRUD operations, import/export, and settings configuration, all with complete reports on the user base.
- Detailed Question Feedback: Generate comprehensive explanations for correct and incorrect answer choices, aligned to PMI mindset.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust, reflecting Inovexio's corporate identity.
- Background color: Light blue-gray (#E8EAF6) to provide a calm, focused learning environment.
- Accent color: A vibrant shade of purple (#7E57C2), analogous to the primary, to highlight CTAs.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern reading experience.
- Use clear, professional icons to represent exam components and functions, enhancing UX and understanding.
- Implement a clean, structured layout to facilitate ease of navigation and focus, ensuring a professional and user-friendly design. A simple interface is crucial to an effective learning environment.
- Subtle animations and transitions will be used to give feedback on actions (correct/incorrect) and provide clear UI cues without distracting from learning.
